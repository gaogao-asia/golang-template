# Step to run
### RUN via local
1. Init
```
$ make init
```

2. Run
```
$ make run
```

### RUN via vscode
1. Init
```
$ make init
```
2. Run
`Mac: cmd + f5`

### RUN via container
1. Init
```
$ make init
```

2. Build
```
$ make quickbuild
```

3. Run
```
$ make run/container
```
